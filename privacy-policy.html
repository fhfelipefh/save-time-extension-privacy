Política de Privacidade da Save Time Extension

Esta política descreve, de forma objetiva, como a Save Time Extension trata as informações armazenadas para viabilizar automações configuradas pelo próprio usuário.

Última atualização: 19/06/2026

1. Introdução

A Save Time Extension é uma extensão de navegador criada para automatizar preenchimentos de formulários e tarefas repetitivas em páginas da web, conforme regras configuradas pelo próprio usuário.

Esta política explica quais informações podem ser armazenadas pela extensão, para qual finalidade elas são utilizadas e quais controles permanecem sob responsabilidade do usuário.

2. Dados armazenados

A extensão pode armazenar localmente no navegador as configurações criadas pelo usuário, incluindo sites habilitados, regras de preenchimento, seletores de campos, tipos de automação e valores definidos pelo usuário para preenchimento automático.

Dependendo das regras criadas pelo usuário, esses valores podem conter informações pessoais, como nome, e-mail, telefone, endereço, cidade, profissão, datas ou outros dados digitados voluntariamente pelo usuário.

3. Finalidade do uso dos dados

Os dados são usados exclusivamente para permitir o funcionamento da extensão, incluindo salvar regras por site e reaplicá-las em páginas compatíveis quando o usuário acessar novamente esses sites.

A Save Time Extension não utiliza essas informações para perfilamento, publicidade, enriquecimento de base de dados ou qualquer outra finalidade alheia à automação configurada pelo próprio usuário.

4. Armazenamento local

As informações são armazenadas localmente no navegador por meio dos mecanismos de armazenamento do Chrome. Isso significa que os dados permanecem no ambiente do navegador do próprio usuário, vinculados ao uso da extensão.

A extensão não envia essas informações para servidores externos, não mantém banco de dados remoto e não depende de backend para guardar regras ou valores de preenchimento.

5. Compartilhamento de dados

A extensão não vende, transfere, compartilha ou divulga dados do usuário para terceiros.

Também não existe sincronização com servidores próprios, repasse comercial de dados ou integração destinada ao compartilhamento das informações configuradas pelo usuário.

6. Coleta automática

A extensão não coleta histórico de navegação, não realiza rastreamento para publicidade, não utiliza analytics externo e não monitora a atividade do usuário para finalidades não relacionadas ao funcionamento da extensão.

Qualquer leitura de página ocorre apenas para permitir a configuração e a execução das automações desejadas pelo usuário nos sites onde ele optar por utilizar a extensão.

7. Permissões do navegador

A extensão utiliza permissões do navegador para identificar a aba atual, acessar páginas em que o usuário deseja configurar automações, salvar regras localmente e aplicar os preenchimentos configurados pelo usuário.

  • Identificar a aba atual e o site em uso para associar regras ao domínio correto.
  • Acessar o conteúdo das páginas nas quais o usuário deseja configurar automações.
  • Salvar localmente regras, seletores, tipos de automação e valores definidos pelo usuário.
  • Aplicar preenchimentos automáticos ou ações configuradas pelo usuário em páginas compatíveis.

8. Controle do usuário

O usuário pode remover ou alterar suas regras a qualquer momento pela interface da extensão.

O usuário também pode excluir os dados da extensão removendo a extensão do navegador ou limpando os dados armazenados pelo navegador. Como o armazenamento é local, o controle sobre permanência e exclusão dessas informações permanece com o próprio usuário.

9. Alterações nesta política

Esta política pode ser atualizada para refletir mudanças na extensão ou em seus recursos.

Quando houver alterações relevantes, a data de atualização deste documento será revisada para indicar a versão mais recente da política publicada.
